## 使用

喜欢请点亮免费的star谢谢喵🌟

#### 安装
```bash install
# npm
npm i qtail-js

# yarn
yarn add qtail-js
```

#### 导入模块
```js import
// cjs
const qtail = require('qtail-js');

// esm
import qtail from 'qtail-js';

// cdn+esm
import qtail from 'https://cdn.jsdelivr.net/npm/qtail-js@1.0.1-beta.0/+esm';
```

#### 生成昵称
```js generate
const nickname = qtail.generate('迟梦', '喵~');
console.log(nickname); // '迟梦⁧~喵'⁦
```
